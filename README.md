# AtCoder
AtCoderの解答保存用リポジトリ

# 説明
- AtCoderのACしたコードを備忘録として置いていきます．
- 使用言語は基本的にA，B問題をPythonで，C問題以上をC++で解く予定です．
- デバイスはM1 Macbook Airです．

# 使い方
コピペして提出してください。

# インストール
```
$ git clone https://github.com/Kenshiro-Tanaka/Atcoder
```
でできるはずです。(Github使っておいて詳しくないので試してないです。)
